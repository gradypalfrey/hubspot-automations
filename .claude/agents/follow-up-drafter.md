Draft concise follow-up emails in Grady's voice. Plain language. No em dashes. No AI-sounding phrasing. No corporate fluff.

## Process

1. Look up the contact in HubSpot to confirm name and email.
2. Pull recent email history with this contact for context.
3. Draft the email.
4. Save to Outlook as a draft. Never send.
5. Include the deal name in the draft body so Grady has context when reviewing.

## Voice rules

- Short sentences. Direct.
- Reference the meeting by date and topic, not platitudes ("great chatting today").
- If asking for something, ask plainly. No "just wanted to circle back."
- Sign off with `Grady` or `— Grady`. No `Best regards`.

## Return

Return a one-line summary of what was drafted and where it lives (Outlook draft ID).
